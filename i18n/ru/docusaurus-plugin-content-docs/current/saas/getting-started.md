SaaS доступен на [https://gorizond.negash.ru/dashboard](https://gorizond.negash.ru/dashboard). Здесь — как зарегистрироваться, что создаётся автоматически, текущие ограничения бесплатного тарифа и как пополнение биллинга через UI снимает лимит в 1 воркер. Отдельно позже будет описано self-hosted.

![Лендинг с кнопкой Go to app](/img/saas/landing-hero.png)

## Цены {#pricing}
- Бесплатно: кластеры Kubernetes API с лимитом **1 worker**; оператор сейчас лишь предупреждает при превышении.
- Платно: пополните биллинг в UI, лимит 1 worker снимается; списания идут с пополненного биллинга при работе кластеров в платном режиме (действуют лимиты плана).

## Что получаете сейчас
- Самостоятельная регистрация с мгновенным доступом в консоль.
- Для каждого нового пользователя создаётся изолированный workspace с правами **admin** на уровне Fleet workspace.
- Запись биллинга создаётся в UI (можно несколько на workspace). Пока биллинг не пополнён, кластеры Gorizond в этом workspace работают по бесплатному плану.
- Бесплатный тариф: можно создавать Kubernetes API кластеры с лимитом **1 worker**. Сейчас при добавлении лишних воркеров оператор только пишет предупреждение; автоматическое отключение пока не включено.

## Регистрация
1) Перейдите на [https://gorizond.negash.ru/dashboard](https://gorizond.negash.ru/dashboard) и нажмите **Sign up with GitHub**.
2) Завершите вход через GitHub — вы попадёте в свой workspace с правами администратора.
3) У каждого пользователя свой workspace. Уже можно приглашать участников или целые GitHub-группы в workspace.

![Форма входа для локальных пользователей](/img/saas/saas-login.png)

## Создание и управление кластерами
- Создавайте кластеры через UI Gorizond: [https://gorizond.negash.ru/dashboard/c/_/gorizond/provisioning.gorizond.io.cluster](https://gorizond.negash.ru/dashboard/c/_/gorizond/provisioning.gorizond.io.cluster). На бесплатном плане держитесь лимита **1 воркер** до пополнения биллинга.
- Можно также открыть приложение **Gorizond** в Rancher (Global Apps → Gorizond), чтобы видеть и управлять кластерами внутри namespace workspace.

![Gorizond clusters UI](/img/saas/saas-gorizond-clusters-table.png)

### Проверка биллинга
- В том же расширении откройте **Billings**, чтобы увидеть объекты биллинга workspace. Balance показывает текущий баланс/долг, **Top Up** запускает платёж и открывает YooKassa.

![Расширение Gorizond — список биллингов](/img/saas/saas-gorizond-billings-table.png)

### Получить платёжную ссылку
- Нажмите **Top Up** в таблице биллинга. Введите сумму и нажмите **Top up on Yookassa** — откроется YooKassa для оплаты.

![Диалог пополнения биллинга](/img/saas/saas-gorizond-billing-topup.png)

## Привяжите биллинг, чтобы снять лимит
После успешной оплаты через YooKassa workspace переключается в платный режим и лимит в 1 воркер снимается (остальные лимиты зависят от плана).

Шаги:
1) В UI (Billing) запросите ссылку на оплату и введите сумму.
2) Перейдите в YooKassa и оплатите.
3) После оплаты биллинг пополняется и привязывается к вашему workspace; лимит в 1 воркер снимается.
4) Создавайте или масштабируйте кластеры без ограничения в 1 воркер (действуют лимиты плана).
